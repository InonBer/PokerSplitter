Screen Flow

The 5 screens of PokerSplitter and how they connect

Screen 1

Home

List of past games (date, players, status). "New Game" button. Tap a past game to view its detail.

↓ New Game
Screen 2

Game Setup

Add player names + initial buy-in amounts. Add as many players as needed. Tap "Start Game".

↓ Start Game
Screen 3

Active Game

Player list with total amount in. Buttons per player: Rebuy, Cash Out. Running totals visible. "End Game" button when ready.

↓ End Game
Screen 4

Final Chip Count

Enter final chip count for each player still in the game. App validates total chips = total money in.

↓ Calculate
Screen 5

Settlement

Clear list: "Dave pays Sarah $30, Mike pays Dave $15..." Minimum transfers algorithm. Share button to send via WhatsApp/iMessage. Saved to history.

Screen 1 also lets you tap any past game → view its Setup, Active log, and Settlement result (read-only).
